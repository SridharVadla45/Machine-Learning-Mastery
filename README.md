## ðŸ› ï¸ Setup & Installation

### Prerequisites

- **Python 3.10+**
- **uv** (modern Python package manager)
- **VS Code** (recommended) or any code editor
- **Git** for version control

### Quick Start

```bash
# Install uv (if not already installed)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Clone/navigate to the repository
cd machine-learning-mastery

# Create environment and install dependencies
uv sync

# Test your setup
uv run 00-python-fundamentals/examples.py

# Start learning!
```

### Running Modules

```bash
# Run examples
uv run 01-linear-algebra/examples.py

# Do exercises
uv run 07-supervised-learning/exercises.py

# Check solutions
uv run 07-supervised-learning/solutions.py
```

See `env-setup.md` for detailed installation instructions.
